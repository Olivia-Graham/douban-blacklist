**豆瓣一键拉黑**

一个很弱智的脚本，帮你在豆瓣批量拉黑/解除拉黑。


**能干什么**

在以下页面会出现一个橙红色按钮，点一下自动处理所有人：

小组贴子的评论页 → 拉黑所有评论的人
小组贴子的点赞页 → 拉黑所有点赞的人
小组贴子的转发页 → 拉黑所有转发的人
小组成员页 → 拉黑所有成员
黑名单页（/contacts/blacklist） → 一键解除所有拉黑（大赦天下）


**注意**

每拉黑一个人会随机等 2~4 秒，是为了不被豆瓣封号，不是卡住了
如果中途触发验证码会自动暂停，手动完成验证码后重新点按钮继续
大赦天下完成后页面会自动刷新
